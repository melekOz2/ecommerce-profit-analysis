## 📌 Business Problem

E-commerce businesses often focus on revenue, but fail to monitor profitability.

This project identifies:
- Loss-making products
- Cost drivers
- Profitable categories

---

# 📊 E-commerce Profit Analysis

This project analyzes e-commerce revenue, cost, and profit performance using SQL and Tableau.


---

## 🚀 Project Overview

The goal of this project is to understand:

- Which products are generating profit
- Which products are causing losses
- How costs impact overall profitability

---

## 📊 Dashboard Insights

- Total Revenue: 14,175
- Total Cost: 14,241
- Total Profit: -65.63 (Loss)

### Key Findings:

- Some categories generate profit (Fashion, Electronics)
- Others consistently lose money (Stationery, Home)
- High shipping and ad costs significantly reduce margins
